# PrevisaoDoTempo
Site feito para estudo de web com IA e uso de API's através do canal DevClub
---
**Passos para clonar e testar o sistema**
1. Clone o repositório para o seu VScode
2. Crie o arquivo **config.php**
  - Acesse os sites **https://openweathermap.org/api** e **https://groq.com/**
  - Faça login e crie uma chave de API nos dois
3. Acesse o arquivo config.php e escreva o seguinte código completando com as suas chaves de API:
```
<?php 
    define('chave', 'suaChaveApiOpenWeather');
    define('chaveGroq', 'suaChaveApiGroq');

?>
```
4. Acesse o seu localhost e teste o sistema, **Obrigado pelo apoio!!!**
